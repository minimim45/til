| 演算子 | 意味 |
| :--- | :--- |
| == | 左右が等しいとき正しい |
| != | 左右が異なるとき正しい |
| > | 左が右より大きいとき正しい |
| >= | 左が右等しいか大きいとき正しい |
| < | 左が右より小さいとき正しい |
| <= | 左が右等しいか小さいとき正しい |
